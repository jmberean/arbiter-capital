Here are the core details and logistics for the ETHGlobal Open Agents hackathon, purely focusing on the event itself. 

### Event Overview
* **Name:** ETHGlobal Open Agents
* **Dates:** April 24 – May 6, 2026 (The hackathon is currently active)
* **Format:** Fully Asynchronous (Anywhere)
* **Cost:** Free to participate
* **Total Prize Pool:** $50,000+ 

### The Core Theme
The overarching focus of this hackathon is building **AI agents that live onchain**. ETHGlobal is specifically looking for projects that move beyond standard off-chain chatbots. The goal is to build autonomous systems capable of execution, agent-to-agent negotiation, and utilizing infrastructure that plugs directly into Ethereum.

### Sponsors & Specific Bounties
The prize pool is distributed across four distinct infrastructure partners, each looking for specific technical implementations:

**1. 0G ($15,000 Prize Pool)**
* **Tech Profile:** The largest AI Layer 1 ecosystem, offering a modular, decentralized AI Operating System (DeAIOS) designed for infinite horizontal scalability to handle massive AI workloads.
* **Goal:** Bringing AI fully on-chain securely and scalably. *(Specific prize distribution details were listed as "coming soon").*

**2. Uniswap Foundation ($5,000 Prize Pool)**
* **Tech Profile:** The leading decentralized exchange protocol. 
* **Goal:** Advancing DeFi through protocol innovation and developer success, specifically highlighting recent developments like Unichain and Uniswap v4. *(Specific prize distribution details were listed as "coming soon").*

**3. Gensyn ($5,000 Prize Pool)**
* **Tech Profile:** Decentralized compute protocol for ML. They are highlighting their **AXL** (Agent eXchange Layer) peer-to-peer network node. AXL provides an encrypted, serverless, decentralized communication layer allowing agents to talk via `localhost`.
* **Bounty:** "Best Application of Agent eXchange Layer (AXL)"
    * 1st Place: $2,500 | 2nd Place: $1,500 | 3rd Place: $1,000
* **Requirements:** Must use AXL for inter-agent/inter-node communication (no centralized message brokers). Must demonstrate cross-node communication.
* **Suggested Tracks:** "Agent Town" (multi-agent simulations with personalities) or Decentralized Agent Messaging platforms.

**4. KeeperHub ($5,000 Prize Pool)**
* **Tech Profile:** The execution and reliability layer for AI agents. They provide guaranteed onchain execution, gas optimization, and retry logic so agents can autonomously move value without failed transactions.
* **Bounty 1:** "Best Use of KeeperHub" ($4,500 Pool)
    * 1st Place: $2,500 | 2nd Place: $1,500 | 3rd Place: $500
    * *Focus Area 1 (Innovative Use):* Solve a real problem using KeeperHub's execution layer via MCP or CLI.
    * *Focus Area 2 (Integration):* Build bridges/plugins for KeeperHub targeting payment rails (x402/MPP) or agent frameworks (ElizaOS, LangChain, etc.).
* **Bounty 2:** "Builder Feedback Bounty" ($500 Pool)
    * Up to 2 teams win $250 each for providing specific, actionable feedback on UX friction, reproducible bugs, or documentation gaps encountered while using KeeperHub during the event.
